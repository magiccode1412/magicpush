ProxyGateway

MagicPush 专用转发服务

📖 项目用途

本服务专为 MagicPush 项目提供，部署在支持双栈 IP 协议的边缘节点上，用于解决不同 IP 协议之间的跨协议访问问题。

由于边缘节点同时支持 IPv4 和 IPv6 双栈协议，本代理可实现双向转发：

  • IPv4 用户访问 IPv6-only 服务器
  • IPv6 用户访问 IPv4-only 服务器

⚡ 工作原理

用户请求 → 边缘节点 (双栈) → 按需回源 (IPv4/IPv6) → 您的服务器
