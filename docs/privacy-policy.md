# 隐私政策

**最后更新日期：2026年3月8日**

## 1. 引言

欢迎使用 **MagicPush（魔法推送）**。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。请您在使用本服务前仔细阅读本政策。

**重要提示：MagicPush 是一款开源的消息推送工具软件，本隐私政策仅适用于由项目开发者官方部署的服务实例。如果您使用的是第三方自行部署的服务实例，请以该部署方的隐私政策为准。**

## 2. 信息收集

### 2.1 我们收集的信息

为了提供推送服务，我们可能收集以下信息：

- **账户信息**：用户名、密码（加密存储）、注册时间
- **渠道配置信息**：您配置的各类推送渠道的令牌、密钥、Webhook地址等
- **推送记录**：推送时间、推送状态、消息标题（不含完整消息内容）
- **系统日志**：错误日志、访问日志用于问题排查

### 2.2 我们不收集的信息

- 消息的完整内容（仅记录标题用于历史查询）
- 您的身份证号、银行卡等敏感个人信息
- 您的通讯录、相册等设备权限信息

## 3. 信息使用

收集的信息将用于：

1. 提供消息推送服务
2. 保障系统安全和稳定性
3. 改善用户体验和服务质量
4. 响应用户的技术支持请求

## 4. 信息存储与安全

### 4.1 数据存储

- 所有用户数据存储在您部署的服务器本地数据库中
- 数据库采用 SQLite 或 MySQL 存储，支持加密存储
- 项目开发者无法访问您自行部署实例中的任何数据

### 4.2 安全措施

- 密码采用 bcrypt 加密存储
- 认证令牌采用 JWT 机制
- 所有敏感配置信息加密保存
- 支持 HTTPS 安全传输

## 5. 信息共享

**我们不会将您的信息出售、出租或共享给任何第三方。**

但以下情况除外：

1. 您主动配置的第三方推送渠道（如企业微信、Telegram 等），消息内容将通过相应平台发送
2. 法律法规要求或政府机关依法要求
3. 经您明确同意的其他情况

## 6. 推送内容免责声明

**重要声明：**

1. **MagicPush 仅作为消息推送的工具软件**，不具备内容审核、过滤或监控功能
2. **推送内容由用户自行创建和发送**，项目开发者无法控制、预审或干预用户发送的内容
3. **项目开发者对用户发送的任何内容不承担法律责任**，包括但不限于：
   - 违法违规内容
   - 侵权内容
   - 骚扰信息
   - 虚假信息
   - 其他违反法律法规或道德准则的内容
4. 用户应对自己发送的内容承担全部法律责任

## 7. 用户权利

您对您的个人信息享有以下权利：

- **访问权**：您可以随时查看自己的账户信息和渠道配置
- **更正权**：您可以随时修改、更新您的个人信息
- **删除权**：您可以删除您的账户及相关数据
- **导出权**：您可以导出您的推送历史记录

## 8. 第三方服务

本服务支持对接以下第三方推送渠道，这些服务由各自的运营方提供，受其各自的隐私政策约束：

- 企业微信（腾讯）
- Telegram
- PushPlus
- WxPusher
- 飞书（字节跳动）
- 钉钉（阿里巴巴）
- 微信公众号（腾讯）
- Server酱
- 自定义 Webhook

使用这些服务时，请同时阅读并遵守相应平台的隐私政策和服务条款。

## 9. 未成年人保护

本服务不面向未满14周岁的未成年人提供。如果您发现我们无意中收集了未成年人的个人信息，请联系我们，我们将尽快删除相关信息。

## 10. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在项目主页公布，重大变更将通过适当方式通知用户。继续使用本服务即表示您接受更新后的政策。
