# 服务条款

**最后更新日期：2026年3月8日**

## 1. 服务说明

**MagicPush（魔法推送）** 是一款开源的消息推送工具软件，支持多种消息渠道的统一推送服务管理。本服务条款（以下简称"本条款"）适用于您使用 MagicPush 软件及相关服务。

**重要说明：MagicPush 是开源软件，本项目仅提供软件代码和工具，不对任何第三方自行部署的服务实例承担责任。如您使用的是第三方部署的服务，请以该部署方的服务条款为准。**

## 2. 服务性质

### 2.1 工具属性

MagicPush 是一个**纯工具类软件**，其核心功能是：

- 提供消息推送的技术通道
- 整合多种第三方推送渠道
- 提供标准化的 REST API 接口
- 管理推送接口和渠道配置

### 2.2 非内容服务

**MagicPush 不提供任何内容服务，具体表现为：**

1. 不生成、编辑、审核或推荐任何推送内容
2. 不存储消息的完整内容（仅记录标题用于历史查询）
3. 不监控、审查用户的推送行为
4. 不对推送内容进行任何形式的干涉
