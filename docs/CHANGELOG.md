# 更新日志 (Changelog)

## [1.2.0] - 2026-03-03

### 新增 (Features)
- 新增微信公众号渠道支持（模板消息推送，支持测试号）
- 新增Server酱渠道支持（微信推送服务）

---

## [1.1.0] - 2026-03-03

### 新增 (Features)
- JWT_SECRET 支持自动生成，首次运行时自动创建安全密钥并存入数据库
- 移除不安全的环境变量默认值
- 简化 Docker 部署配置，JWT_SECRET 变为可选项

---

## [1.0.1] - 2026-02-15

### 修复 (Fixes)
- 修复用户登录权限验证问题 - 认证中间件未传递 role 信息
- 修复数据库初始化逻辑 - 应用启动时自动初始化数据库

### 优化 (Optimizations)
- 添加集中式版本管理系统
- 实现版本号自动同步到所有相关文件
- 新增版本信息 API 接口

---

## [1.0.0] - 2026-02-15

### 新增 (Features)
- 初始版本发布
- 支持企业微信、Telegram、PushPlus、WxPusher、飞书、钉钉等消息渠道
- Webhook 通用推送支持
- 用户认证与权限管理（JWT 双令牌机制）
- 渠道配置与推送接口管理
- 推送历史记录与状态追踪
- 响应式 Web 管理界面
- 深浅色主题切换
- 配置导入导出功能

---

## 版本说明

版本号遵循 [语义化版本 (Semantic Versioning)](https://semver.org/lang/zh-CN/) 规范。

- **主版本号 (Major)**：不兼容的 API 修改
- **次版本号 (Minor)**：向下兼容的功能性新增
- **修订号 (Patch)**：向下兼容的问题修正
